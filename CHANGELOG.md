# Changelog

## 0.2.0 (2026-07-04)

### Features

- Serve module configuration from the backend with public-safe deployment defaults.
- Add dedicated Licenses and Permissions pages.
- Add SQLAlchemy persistence foundation, Alembic migrations, and indexed inventory tables.
- Scope overview widget preferences by backend-served deployment config.
- Add collector framework skeleton for background inventory sync.
- Add Services and VMs repositories for ownership, status, and review queries.
- Improve dashboard layout controls with widget ordering, widget sizing, table search, and empty states.

### CI/CD

- Restrict release asset uploads to frontend bundles and backend distribution files.

## 0.1.0 (2026-07-04)

### Features

- Bootstrap Spaghetti Desk as a public-safe DevOps Control Center.
- Add FastAPI backend demo inventory APIs.
- Add React operations dashboard with configurable in-app module views.
- Add Docker Compose local runtime, docs, tests, and CI scaffolding.
